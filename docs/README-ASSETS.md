# README Assets Checklist

This file lists the assets referenced in the main `README.md`. Please generate or record these assets and place them in the `docs/assets` directory (or update the links in `README.md`).

## Required Images / GIFs

| Asset Name | Description | Location | Status |
| :--- | :--- | :--- | :--- |
| `demo-preview.gif` | A 10-15 second autoplay GIF showing the dashboard interactivity (hovering over charts, scrolling). | `docs/assets/demo-preview.gif` | ⬜ Pending |
| `pipeline-architecture.png` | An infographic or flowchart showing the automation pipeline (Teams -> Power Automate -> Gemini -> Make). | `docs/assets/pipeline-architecture.png` | ⬜ Pending |
| `dashboard-screenshot.png` | A high-resolution screenshot of the full dashboard. | `docs/assets/dashboard-screenshot.png` | ⬜ Pending |

## Instructions for Generation

1.  **Demo GIF:**
    *   Open `index.html` in a browser.
    *   Use a screen recorder (e.g., LICEcap, ScreenToGif).
    *   Record scrolling down the page, hovering over the "Age Distribution" and "Knowledge" charts.
    *   Save as `demo-preview.gif`.

2.  **Pipeline Infographic:**
    *   Use Mermaid.js (included in README) or a tool like Canva/Figma.
    *   Visual flow: User Registration (Teams) -> Data Cleaning (Power Automate) -> Analysis (Gemini) -> Dashboard (Web) -> Confirmation (Make.com).

3.  **Screenshots:**
    *   Use browser developer tools (Ctrl+Shift+I) -> "Capture full size screenshot".
